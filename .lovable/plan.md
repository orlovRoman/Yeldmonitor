

# План: Восстановление панели алертов на дашборде

## Задача
Вернуть панель "Алерты" на главную страницу, сохранив панель "Новые пулы" и "Снижение Implied APY".

## Текущее состояние
- **ImpliedDropsPanel** - показывает только снижения Implied APY за последний час
- **NewPoolsPanel** - показывает новые пулы за 24 часа
- **AlertsPanel** - компонент существует и полностью готов, но не используется

## Изменения

### 1. Обновить Index.tsx

Добавить импорт и отображение `AlertsPanel`:

```
src/pages/Index.tsx
├── Импортировать AlertsPanel
└── Добавить в сетку дашборда
```

**Новая структура макета:**
- Первый ряд: карточки статистики (без изменений)
- Второй ряд: три колонки
  - Слева: Алерты (полный список всех алертов)
  - По центру: Снижение Implied APY (за последний час)
  - Справа: Новые пулы (за 24ч)

## Технические детали

Изменения в `src/pages/Index.tsx`:

1. Добавить импорт:
```typescript
import { AlertsPanel } from '@/components/dashboard/AlertsPanel';
```

2. Изменить сетку с 2 на 3 колонки:
```tsx
<div className="grid grid-cols-1 lg:grid-cols-3 gap-6">
  <AlertsPanel />
  <ImpliedDropsPanel />
  <NewPoolsPanel />
</div>
```

## Результат
Дашборд будет отображать три панели:
- **Алерты** - полный список всех активных алертов с AI-анализом
- **Снижение Implied APY** - быстрый обзор падений за час  
- **Новые пулы** - недавно добавленные пулы
